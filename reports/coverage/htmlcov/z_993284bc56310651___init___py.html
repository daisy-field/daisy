6"""Collection of executable scripts to start up the various federated IDS components 

7of the daisy package, more or less pre-configured by the respective script for ease 

8of use and demonstration purposes. Each module contains the setup of a single generic 

9F-IDS component, i.e. script to be started, either in separate python instances or 

10via threads. Alternatively one can also launch these scripts directly via the command 

11line. Most of these components require other components however to form a functioning 

12systems, e.g. demo setups require specific demo components as well. See the 

13docstrings of the respective demos. 

14 

15Currently, the following generic federated IDS components are provided: 

16 

17 * dashboard - Auxiliary dashboard for all other servers' aggregated results. 

18 * eval_aggr_server - Auxiliary aggregation server for evaluation metric results. 

19 * model_aggr_server - Federated model aggregation server using federated averaging. 

20 * pred_aggr_server - Auxiliary aggregation server for prediction value results. 

21 

22Author: Fabian Hofmann 

23Modified: 17.06.24 

24"""
